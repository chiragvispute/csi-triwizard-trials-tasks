ðŸ”¹ 3) Recursion

What: A function can call itself to solve smaller parts of a problem.

Example:

def countdown(n):
    if n == 0:
        print("Done!")
    else:
        print(n)
        countdown(n - 1)

countdown(3)  # 3, 2, 1, Done!

ðŸ”¹ 4) Base Case in Recursion

What: Every recursive function needs a stopping condition to avoid infinite calls.

Example:

def factorial(n):
    if n == 0:
        return 1   # base case
    return n * factorial(n - 1)

print(factorial(4))  # 24
