ðŸ”¹ 1) Indexing and Negative Indexing

What: Elements in a list can be accessed using indexes. Positive indexes start from 0, and negative indexes access elements from the end (-1 is last element).

Example:

arr = [1, 2, 3, 2, 1]
print(arr[0], arr[-1])  # 1, 1

ðŸ”¹ 2) Loops (for)

What: Loops help in iterating from both ends to compare elements.

Example:

arr = [1, 2, 3, 2, 1]
for i in range(len(arr) // 2):
    print(arr[i], arr[-(i+1)])

ðŸ”¹ 3) Conditional Statements (if)

What: Used to check if two values are equal or not during iteration.

Example:

x, y = 5, 5
if x == y:
    print("Equal")

ðŸ”¹ 4) Built-in Functions (Optional Shortcuts)

What: Python provides built-ins like len() for length, reversed() for reverse iteration, and slicing ([::-1]) to reverse a list or string.

Example:

arr = [1, 2, 3, 2, 1]
print(len(arr))         # 5
print(list(reversed(arr)))  
print(arr[::-1])        # [1, 2, 3, 2, 1]
